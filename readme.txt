✈️ DJ Travels Website

A fully responsive travel website developed using HTML5 and CSS3.
This project focuses on clean UI design, structured layout, and responsive web development principles.

🚀 Features

Hero section with optimized background image positioning

Service cards with hover effects

About section with side-by-side layout

Responsive image gallery

Clean navigation bar and footer

Mobile-friendly design using Flexbox

🛠️ Technologies Used

HTML5

CSS3

Flexbox

Google Fonts

🎯 Purpose

This project was built to strengthen frontend development skills and practice responsive layout structuring.
